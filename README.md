# Ratish Oberoi Portfolio

World-class AI Infrastructure Engineer portfolio for Ratish Oberoi.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Mermaid
- GitHub REST API client-side widgets

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Static Build

```bash
npm run build
```

## GitHub Pages Build

```bash
npm run build:github
```

The site exports to `out/` and is deployable through GitHub Pages Actions.
